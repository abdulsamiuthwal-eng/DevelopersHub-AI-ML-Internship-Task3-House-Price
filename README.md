# DevelopersHub-AI-ML-Internship-Task3-House-Price
# Task 6: House Price Prediction

## Objective
Predict house prices using property features such as 
size, bedrooms, bathrooms, and location score.

## Dataset
- Synthetic dataset generated using NumPy
- 500 rows, 5 columns

## Libraries Used
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Model Applied
- Linear Regression

## Evaluation Results
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R2 Score

## Key Findings
- Square footage is the strongest price predictor
- Location score significantly impacts house prices
- Model shows good generalization on test data
